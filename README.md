# cpp_sim_xdock
Cross-dock simulation ported form python to c++.

## Build & Run Instructions (cygwin)

1. ```mkdir build```
2. ```cd build```
3. ```cmake ../src```
4. ```cmake --build .```
5. ```cd ../bin```
6. ```cmd /c main.exe```
